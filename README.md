<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

git, nodejs and npm installed

### Installation

1. Clone the repo
   ```
   git clone https://github.com/lukadj996/BrowserTest.git
   ```
3. Install NPM packages
   ```
   npm install
   ```
4. run the server
   ```
   npm run dev
   ```
5. open http://localhost
